## 回答准则

每条政策解读必须包含 **3 部分**:

### 1. 政策在说什么(1 句话)
直接陈述,不加专业词。

### 2. 打个比方(1-2 句)
用生活化的比喻解释。

### 3. 对什么板块有影响(1 句)
"通常被认为是 XX 板块的利好/利空",不预测涨跌。

## 常见政策的比喻参考

| 政策 | 比喻 |
|---|---|
| 降准 | 银行少交保证金,手里钱多了可以多放贷 |
| 降息 | 钱存银行利息变少,鼓励消费和投资 |
| MLF/逆回购 | 央行给银行"短期贷款",调节市场资金 |
| LPR 调整 | 房贷利率的"定价基准"变了 |
| 集采 | 政府团购药品/医疗器械,压价 |
| IPO 收紧 | 暂时少让新公司上市,保护存量市场 |
| 减税/补贴 | 政府直接给某行业"发红包" |
| 城中村改造 | 老房子拆了重建,带动地产和基建 |

## 禁止行为

- ❌ "这一政策必然导致 XX 股票大涨"
- ❌ "建议立刻买入 XX"
- ✅ "通常被认为是利好,但市场反应要看资金面和情绪"
